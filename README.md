# Influencer Rate Card Studio — Germany & India 2025

A fully interactive influencer rate card tool with live rate calculation for the German 🇩🇪 and Indian 🇮🇳 markets.

## Features
- Dynamic rate calculator (followers + engagement rate + niche)
- Germany (€) and India (₹) market benchmarks
- 3 pricing tiers: Standard, High-Value, Max Revenue
- Market-benchmarked rates with your personalised column highlighted
- Add-on multiplier guide (usage rights, exclusivity, retainers)
- Legal/tax compliance notes for both markets

## Deploy on Vercel

### Option 1 — Vercel Dashboard (easiest)
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **Add New Project**
3. Import your GitHub repo
4. Leave all settings as default → click **Deploy**

### Option 2 — Vercel CLI
```bash
npm i -g vercel
vercel
```

## Local Development
Just open `index.html` in any browser — no build step needed.
